# About

I'm a data scientist and ML engineer who carries problems from raw data all the way to a working, evaluated result. My portfolio spans the field — a multi-architecture generative AI pipeline, a productized generative-AI service I shipped to a live product, a cross-platform AI scheduling assistant, panel-data econometrics, and scripted scientific visualization. The throughline across all of them is rigor: clean pipelines, real evaluation metrics, and code I can defend.
