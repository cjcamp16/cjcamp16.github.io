# cjcamp16.github.io

Personal portfolio site for Cole Campbell — data scientist and ML engineer.

Live at **https://cjcamp16.github.io**

Plain HTML, CSS, and a small JS file — no build step. Edit `index.html` / `style.css` and push to update.

## Structure
- `index.html` — all content (hero, about, projects, skills, experience, contact)
- `style.css` — styles
- `main.js` — sticky nav, mobile menu, footer year
- `assets/` — project cards, headshot, résumé PDF
