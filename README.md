# cjcamp16.github.io

Personal portfolio site for Cole Campbell, data scientist and ML engineer.

Live at **https://cjcamp16.github.io**

Plain HTML, CSS, and a small JS file, with no build step. Edit `index.html` or `style.css` and push to update.

## Structure
- `index.html`: all content (hero, about, projects, skills, experience, contact)
- `style.css`: single-accent design system, responsive, reduced-motion aware
- `main.js`: sticky nav, mobile menu, scrollspy, reveal-on-scroll, footer year
- `assets/`: project thumbnails, headshot, resume PDF
