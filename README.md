# cjcamp16.github.io

Personal portfolio site for Cole Campbell, data scientist and ML engineer.

Live at **https://cjcamp16.github.io**

Plain HTML and CSS, no JavaScript, no build step. Edit `index.html` or `style.css` and push to update.

## Structure
- `index.html`: everything (about, projects, skills, experience, contact)
- `style.css`: one small stylesheet
- `assets/`: headshot and resume PDF
